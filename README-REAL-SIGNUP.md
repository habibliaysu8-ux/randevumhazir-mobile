# Gerçek kayıt sürümü

Bu sürümde partner ve müşteri için demo hesap yoktur.

## İlk kullanım
1. Admin paneline admin hesabıyla gir.
2. Partner panelinden gerçek işletme hesabı oluştur.
3. Müşteri panelinden gerçek müşteri hesabı oluştur.
4. Oluşan mağaza, hizmet ve randevular aynı veritabanında saklanır.
